Para executar o algoritmo use o seguinte comando:
python dijkstra.py

O Arquivo de entrada deve seguir exatamente a seguinte estrutura:

<V1>,<V2>,<V3>,<Vn>,
<Dv1v1> <Dv1v2> <Dv1v3> <Dv1vn>
<Dv2v1> <Dv2v2> <Dv2v3> <Dv2vn>
<Dv3v1> <Dv3v2> <Dv3v3> <Dv3vn>
<Dvnv1> <Dvnv2> <Dvnv3> <Dvnvn>

Onde <V1>...<Vn> são os vertices e <Dv1v1>...<Dvnvn> são as distancias entre os vertices.
